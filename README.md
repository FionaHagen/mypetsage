My Pet's Age
=========

An android app to calculate and display the age of your pet.

Requirements:
- Android 2.3+

Features:
- Ability to add multiple pets
- Ability to delete a pet
- Show age in different formats:
  - Years, Months, Weeks, Days
  - Years
  - Months
  - Weeks
  - Days
- Widget (Android 3+)
